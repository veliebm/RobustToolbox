Interface IConfigurationManager

Stores and manages global configuration variables.

Namespace: Robust.Shared.Configuration
Assembly: Robust.Shared.dll
Syntax
public interface IConfigurationManager
Remarks

Accessing (getting/setting) main CVars is thread safe. Note that value-changed callbacks are ran synchronously from the thread using SetCVar(String, Object), so it is not recommended to modify CVars from other threads.
