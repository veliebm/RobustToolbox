Class DependencyAttribute

Specifies that the field this is applied to is a dependency, which will be resolved by IoCManager when the containing class is instantiated.

Inheritance
System.Object
DependencyAttribute
Namespace: Robust.Shared.IoC
Assembly: Robust.Shared.dll
Syntax
public sealed class DependencyAttribute : Attribute
Remarks

The dependency is resolved as if Resolve<T>() were to be called, but it avoids circular references and init order issues due to internal code in the IoCManager.

The dependency can be injected into read only fields without issues, and as a matter of fact it is recommended to use read only fields.

If you would like to run code after the dependencies have been injected, use IPostInjectInit
