Class Rope

A binary tree data structure for efficient storage of large mutable text.

Inheritance
System.Object
Rope
Namespace: Robust.Shared.Utility
Assembly: Robust.Shared.dll
Syntax
public static class Rope : object
Remarks

Read the Wikipedia article, nerd Also read the original paper, it's useful too.

Like strings, ropes are immutable and all "mutating" operations return new copies.

All indexing functions use langword_csharp_long indices. While individual rope leaves cannot be larger than an langword_csharp_int, ropes with many leaves may exceed that.
