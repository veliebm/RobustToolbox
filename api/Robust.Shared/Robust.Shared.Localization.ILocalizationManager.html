Interface ILocalizationManager

Provides facilities to obtain language appropriate in-game text.

Namespace: Robust.Shared.Localization
Assembly: Robust.Shared.dll
Syntax
public interface ILocalizationManager
Remarks

Translation is handled using Project Fluent (https://www.projectfluent.org/) You pass a Fluent 'identifier' as a string and the localization manager will fetch the message matching that identifier from the currently loaded language's Fluent files.
