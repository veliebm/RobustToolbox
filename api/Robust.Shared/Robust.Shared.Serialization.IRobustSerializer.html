SerializeDirect<T>(Stream, T)

Serializes an object with exact known type to skip a type ID header.

Declaration
void SerializeDirect<T>(Stream stream, T toSerialize)
Parameters
Type Name Description
Stream stream

The stream to write into.
T toSerialize

The object to serialize. The object MUST have the exact type of T
Type Parameters
Name Description
T

The type of object to serialize.
Remarks

This is more efficient than Serialize(Stream, Object) because it does not use a few bytes on type ID header. Output from this method is not compatible with Deserialize(Stream) or vice versa. The type argument of the DeserializeDirect<T>(Stream, out T) call must also match, obviously.
