Interface IEntitySystemManager

Like SS13's master controller. It controls IEntitySystem instances. These instances have set of rules for which components they can run over.

Namespace: Robust.Shared.GameObjects
Assembly: Robust.Shared.dll
Syntax
public interface IEntitySystemManager
Remarks

The management of these amounts to a couple of things:

  • Periodically ticking them through Update(Single).
Periodically ticks IEntitySystem instances.
