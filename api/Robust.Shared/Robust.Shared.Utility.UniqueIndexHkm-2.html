Struct UniqueIndexHkm<TKey, TValue>

A mutable dictionary of mutable and immutable sets for use as an index of unique values related to another collection. Imitates the behavior of a write-focused index in a RDBMS.

Implements
IEnumerable<KeyValuePair<TKey, HashSet<TValue>>>

Remarks

Use when the index's keys change more rapidly or when insertion and removal of keys is preferred over read time. It is intended to explicitly construct this index before use. Do not refer to a UniqueIndexHkm<TKey, TValue> by it's interface. See Robust.Shared.Utility.IUniqueIndex`2 for details.
