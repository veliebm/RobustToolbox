Class RefList<T>

Supposedly a high-performance version of , that allows fetching direct references to the underlying contents.

Inheritance
System.Object
RefList<T>
Namespace: Robust.Shared.Utility
Assembly: Robust.Shared.dll
Syntax
public sealed class RefList<T> : IList<T>
Type Parameters
Name Description
T

The type of the contents of the list. This must be an unmanaged type.
Remarks

Due to this type's nature, keeping references to the contents of this list while mutating it is undefined behavior. Don't do it.
