Class Parse

Helpers for parsing culture-invariant data.

Inheritance
System.Object
Parse
Namespace: Robust.Shared.Utility
Assembly: Robust.Shared.dll
Syntax
public static class Parse : object
Remarks

APIs like are culture sensitive by default, and making them not culture sensitive is extremely verbose. These helpers are culture insensitive without requiring you to write a whole shakespeare novel for the privilege of having code that isn't gonna break for French people.
