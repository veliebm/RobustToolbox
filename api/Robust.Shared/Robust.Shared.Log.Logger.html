Class Logger

Static logging API front end. This is here as a convenience to prevent the need to resolve the manager manually.

Inheritance
System.Object
Logger
Namespace: Robust.Shared.Log
Assembly: Robust.Shared.dll
Syntax
public static class Logger : object
Remarks

This type is simply a proxy to an IoC based ILogManager and its sawmills. As such, no methods or properties will work if IoC has not been initialized yet.
