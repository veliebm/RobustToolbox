Target

If not null, this entity is used to update the eye's position instead of just using the component's owner.

Declaration
public EntityUid? Target
Field Value
Type Description
System.Nullable<EntityUid>
Remarks

This is useful for things like vehicles that effectively need to hijack the eye. This allows them to do that without messing with the main viewport's eye. This is important as there are some overlays that are only be drawn if that viewport's eye belongs to the currently controlled entity.
