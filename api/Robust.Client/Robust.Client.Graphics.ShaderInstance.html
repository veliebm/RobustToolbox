Class ShaderInstance

A shader instance is a wrapper around a shader that can be used for rendering. It can contain extra, mutable parameters depending on the shader backing it.

Inheritance

ShaderInstance
Namespace: Robust.Client.Graphics

Syntax
public abstract class ShaderInstance : IDisposable
Remarks

Shader instances are often shared to reduce bloat. As such, they can be made "immutable" to avoid accidentally modifying the shader instance used by every entity on the station.

A shader can be made immutable with MakeImmutable(). This is irreversible. After doing this, operations such as SetParameter(String, Single) will throw a .

You can "duplicate" a shader instance to make a separate, once again mutable, copy with Duplicate().
