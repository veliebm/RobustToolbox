A multi-line text editing control.

Remarks

This control uses a Rope to performantly manipulate editable text.

Text selection and cursor positions are tracked with two values: a "selection start" and a "cursor position". When you do not have a selection (i.e. just a lone cursor) these values are at the same position. If you have an active selection, the cursor position is the part of the selection you can move around (via mouse or shift+left/right), while the start is the part that stays in place always.

Positions inside this control are all in chars. Cursor positions and such should never be inside a surrogate pair, however.

Remarks

This isn't editable by the user, but to make the internals simpler, it is still exposed as a rope.

When set to null, the placeholder style pseudo-class will not be applied when the actual text content is empty. If set to an empty text rope this does happen, but obviously no text is displayed either way.
